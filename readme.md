# Trello API

## Requirements
- nodejs and npm installed

## Setup
1. copy `.npmrc.example` in `.npmrc` file
2. fill the credential tokens

## Run Application
```sh
npm start
```

## Run Test

#### Unit Test
```sh
npm test
```

#### Smoke Test
```sh
npm run test:smoke
```

#### All Test
```sh
npm run test:all
```
